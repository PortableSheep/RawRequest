Quick reference

Script helpers & chaining at a glance.

Apply these helpers inside < (pre) or > (post) script blocks to mutate requests, enrich responses, or fail fast. Logs stream into the floating Script Console to keep feedback in view.

Helper quick reference

  • setVar/getVar keep shared state between requests.
  • setHeader + updateRequest patch outbound traffic after parsing.
  • assert throws when expectations fail, halting chained runs.
  • delay awaits async pauses so you can coordinate external systems.
  • console.log | warn | error stream into the Script Console buffer.

Chaining & variables

Name requests with @name, create dependencies with @depends otherRequest, and reference prior responses using {{request2.response.json.token}}. Variables defined via @foo or @env.prod.token are merged automatically.

Response objects expose response.status, response.headers, and response.json for assertions or downstream prep.

History + cancellation

Every execution is timestamped and replayable in the History sidebar. When a long chain misbehaves, hit Cancel and RawRequest tears down the request via its unique ID.

Combine this with the Script Console to correlate which script emitted a log right before cancellation.
